Introduction

Base URL: http://community.loc

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Register

POST
http://community.loc
/api/auth/register

Headers

Content-Type
Example:
application/json
Accept
Example:
application/json

Body Parameters

phone
string
required

Must match the regex /^(998)([0-9]{9})$/.

Example:
998947672376
email
string

Must be a valid email address.

Example:
gkulas@example.org
name
string
required

Must be at least 2 characters.

Example:
oslaqc
password
string
required

Must be at least 6 characters.

Example:
3\hgPSZ{qRk~P
password_confirmation
string

This field is required when password is present. The value and password must match. Must be at least 6 characters.

Example:
dlfiuzwrutkvesznbhz
Example request: 
const url = new URL(
    "http://community.loc/api/auth/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "phone": "998947672376",
    "email": "gkulas@example.org",
    "name": "oslaqc",
    "password": "3\\hgPSZ{qRk~P",
    "password_confirmation": "dlfiuzwrutkvesznbhz"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
Example response:
{
    "data": {
        "success": true
    }
}
{
    "message": "Unauthorized"
}
